# Contributing

Thanks for your interest in contributing to MyDataCanvas OSS!

## Development

```bash
cd oss
npm install
npm run build
npm run test
```

## Pull requests

- Keep changes scoped to `/oss` only.
- Do not import from private packages or apps outside `/oss`.
- Add/adjust tests when behavior changes.
